# CTI SKB content: public CTI knowledge

The CTI SKB SHALL contain by default public and curated cyber threat intelligence (CTI) content, such as attack campaigns, intrusion sets, and TTPs,
e.g., the MITRE ATT&CK data sets.

### Rationale
Ground knowledge (terminological axioms in DL) of the knowledge base. Provides a global picture of the threat landscape.
