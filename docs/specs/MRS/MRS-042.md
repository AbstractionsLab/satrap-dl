# CyFORT CTI continuous analysis

SATRAP SHALL provide an automation mechanism for ingesting data from the CyFORT CTI repository
and running predefined knowledge derivation functions on the newly ingested data, configurable
by user-set parameters.

### Rationale
For integration of the different components in the CyFORT stack supporting continuous
knowledge derivation, to maintain up to date and relevant CTI.
