# Input validation

SATRAP components receiving input to the system SHALL validate the input and reject it in case the validation fails.
The validation may include integrity checks, syntactic checks, semantic checks, parameter out-of-range checks, etc.

### Rationale
To prevent code injection.

### Acceptance criteria
See validation test case specification
