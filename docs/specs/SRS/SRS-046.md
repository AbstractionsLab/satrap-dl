# CTI analysis toolbox

SATRAP SHALL have a component providing end-user functionality accessible programmatically via the frontend.
This functionality consists primarily of parametrizable high-level CTI queries.

### Rationale
For separation of duties in SATRAP and to support automation of CTI analysis tasks.
